# 新我生活系统手机安装版

这是一个静态 PWA 版本。数据仍然只保存在手机浏览器本地 `localStorage`，没有后端。

## GitHub Pages 安装

1. 把这个文件夹里的所有文件上传到你的 GitHub 仓库。
2. 在仓库 Settings 里开启 Pages。
3. 选择部署分支，例如 `main`，目录选择仓库根目录。
4. 等待 GitHub Pages 生成网址。
5. 用 iPhone Safari 打开那个网址。
6. 点击 Safari 分享按钮。
7. 选择“添加到主屏幕”。
8. 之后从主屏幕打开“新我”，即可像一个手机小应用一样使用。

如果你把这些文件放在仓库根目录，打开地址通常类似：

```text
https://你的用户名.github.io/你的仓库名/
```

## 备份

记录只在当前手机浏览器本地保存。换手机、清理 Safari 数据、换托管域名或换浏览器前，请先在应用里的“备份”页面导出 JSON。

## 离线

第一次通过 HTTPS 打开后，核心页面会被缓存。后续从主屏幕打开时，可以在无网络状态下继续使用已缓存的应用。
